## Setup

Install dependencies with `npm install`.

## Running Webpack Development Server

```sh
npm start
```

## Running Jest Test Framework

```sh
npm test
```

## Running Storybook Visual Testbed

```sh
npm run storybook
```

## Running Cypress Visual Tests

```sh
npm run cypress
```

## Dependencies

- Node 10.x or above
- NPM 5.x or above
- PG 6.x
- axios
- classnames
- dotenv 2.x
  -reactdom
- pg-native 3.x
- storybook
- cypress
- jest
